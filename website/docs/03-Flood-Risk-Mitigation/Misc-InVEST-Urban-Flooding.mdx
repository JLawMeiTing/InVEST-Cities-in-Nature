## Glossary
| Variable              | Abbreviation       | Meaning             | Type   |
| :-------------------- | :------------------| :------------------ | :---:  |
| Biophysical Table     | -                  | Table of curve number data for each Land use/Land cover class| Input  |
| Soil Hydrologic Group | -                  | Map of soil hydrologic groups. Pixels may have values 1, 2, 3, or 4, corresponding to soil hydrologic groups A, B, C, or D, respectively| Input |
| Built Infrastructure  | -                  | Map of building footprints| Input |
| Damage Loss Table     | -                  | Table of potential damage loss data for each building type| Input |
| Runoff                | Q                  | Runoff              | Output |
| Runoff retention (m<sup>3</sup>)               | -                  | runoff retention volume on pixel| Output |
| Service| -           | An indicator of the runoff retention service for the watershed. Only calculated when the Built Infrastructure Vector input is provided | Output |
